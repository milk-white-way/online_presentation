# Build on Strong Foundations

**AMReX** — developed at LBNL for the Exascale Computing Project (ECP)

<v-clicks>

- Block-structured **adaptive mesh refinement**
- Supports cell-centered and face-centered data structures
- Fully portable, scalable, with **GPU support**
- Compatible with widely used libraries (HYPRE, PETSc, etc.)
- Multiple visualization options (yt, Amrvis, ParaView, VisIt)

</v-clicks>
